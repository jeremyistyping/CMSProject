# Dokumentasi Sistem Akuntansi Modern

Dokumentasi ini dibagi menjadi beberapa bagian agar mudah dipahami dan dipelihara. Mulailah dari Overview, lalu lanjut ke Panduan Penggunaan dan Flow Bisnis/Akuntansi sesuai kebutuhan Anda.

Daftar isi:
1. 01_OVERVIEW.md — Ringkasan arsitektur, modul, dan teknologi
2. 02_USAGE_GUIDE.md — Cara menjalankan aplikasi dan alur penggunaan dasar
3. 03_BUSINESS_FLOW.md — Flow bisnis: Sales, Purchase, Inventory, Cash/Bank, Asset
4. 04_ACCOUNTING_FLOW.md — Flow akuntansi dan jurnal otomatis (double-entry)
5. 05_USER_ROLES_AND_FLOW.md — Role pengguna, akses, dan user journey
6. 06_CONFIGURATION.md — Konfigurasi environment & accounting_config.json
7. 07_API_GUIDE.md — Ikhtisar endpoint API dan contoh penggunaan
8. 08_REPORTING.md — Laporan keuangan dan analitik
9. 09_SECURITY_AND_DEPLOYMENT.md — Keamanan, audit, dan deployment
10. 10_BALANCE_MONITORING.md — Monitoring saldo & verifikasi pembukuan
11. 11_INVENTORY_AND_ASSETS.md — Inventory control & asset management
12. 12_DIAGRAMS.md — Diagram flow (Mermaid)

Tips:
- Gunakan kombinasi file ini untuk onboarding tim baru.
- Sesuaikan akun-akun default dan pajak di `backend/config/accounting_config.json`.
- Selaraskan mapping akun PPN sesuai kebutuhan bisnis Anda.
